1. Informações gerais

O Cardio AI é um aplicativo voltado para saúde, bem-estar e monitoramento pessoal, oferecendo lembretes de medicamentos, análise básica de dados fornecidos pelo usuário e recomendações gerais de hábitos saudáveis.

2. Dados coletados

O app coleta apenas dados inseridos diretamente pelo usuário, incluindo:

  • Peso
  • Altura
  • Idade
  • Medicamentos utilizados
  • Preferências de lembretes
  • Informações do questionário inicial

Não solicitamos nome, e-mail, número de telefone ou outras informações pessoais.

3. Uso dos dados

Os dados são usados exclusivamente para:

  • Gerar lembretes personalizados
  • Calcular análises básicas
  • Recomendações gerais de saúde

Não usamos dados para publicidade e não compartilhamos com terceiros.

4. Armazenamento e segurança

Os dados podem ser armazenados localmente no dispositivo e podem ser criptografados em trânsito. Utilizamos boas práticas de segurança para proteger as informações.

5. Compartilhamento de dados

O Cardio AI não compartilha dados com terceiros ou parceiros.

6. Direitos do usuário

O usuário pode:

  • Solicitar a exclusão de dados
  • Parar de usar o app
  • Reinstalar para limpar dados locais

8. Menores de idade
